# Deep Research 
> 学习和借鉴Gemini Deep Research / Deerflow 项目 ，实现一个Deep Research 项目
## 后端
启动后端
```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --host 0.0.0.0 --port 8000
```

## 前端
启动前端
```bash
cd frontend
npm install
npm start
```
